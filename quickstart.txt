AdlerIt Quick Start
===================

AdlerIt computes Adler-32 checksums in a small native Windows desktop app.
Download AdlerIt-Windows-x64.exe and run it. There is no installer and no
command-line interface.

AdlerIt makes no network connections and writes no settings, history, cache, or
data files.


1. First Run
------------

- Download AdlerIt-Windows-x64.exe.
- Run the executable.
- The AdlerIt window opens directly, without a terminal window.
- No installation or PATH setup is required.


2. Using the App
----------------

- Type or paste text into the INPUT box.
- The Adler-32 value updates live, shown as hex (0x...) and decimal, with the
  UTF-8 byte count.
- Click Copy to copy the hex checksum to the clipboard.


3. Appearance
-------------

AdlerIt follows your Windows light/dark theme and uses a fixed blue accent.


4. Verifying Your Download
--------------------------

GitHub shows a SHA-256 digest next to each release asset. Compute the hash of
your download and compare it to that digest:

  Get-FileHash .\AdlerIt-Windows-x64.exe -Algorithm SHA256

Release binaries also carry a GitHub build-provenance attestation, and are
scanned with Microsoft Defender before publication. To verify provenance with
the GitHub CLI:

  gh attestation verify AdlerIt-Windows-x64.exe --repo Joshndroid/AdlerIt

Note: Adler-32 is a checksum for integrity and error detection, not a
cryptographic hash. Do not rely on it to prove authenticity against a malicious
party.


5. Troubleshooting
------------------

The window does not open:
- Make sure you downloaded AdlerIt-Windows-x64.exe for 64-bit Windows.

Different result than expected:
- AdlerIt hashes UTF-8 text exactly as entered, so whitespace and trailing
  newlines change the checksum.
